# Liquidity Terminal - AI-Powered Financial Intelligence Platform

An affordable, professional-grade financial intelligence platform that makes global markets understandable, accessible, and actionable for everyone.

## Features

✅ **Real-time Market Tracking**
- Stocks, indices, commodities, crypto, and bonds
- Live price updates and technical analysis

✅ **AI-Powered Insights**
- Understand why markets move
- Inflation, interest rates, earnings explanations
- Portfolio analysis and scenario simulation

✅ **Advanced Stock Screener**
- Filter by market cap, revenue growth, valuation
- Sector rotation dashboards
- Custom financial metrics

✅ **Portfolio Management**
- Simulate and backtest strategies
- Risk scoring and diversification analysis
- Performance tracking

✅ **Market Intelligence**
- Economic calendar with impact analysis
- News feed and sentiment analysis
- Institutional flow tracking

✅ **AI Assistant**
- Ask financial questions naturally
- Get simplified explanations
- Scenario planning
